# Финальный проект по модулю ООП и работе с API «Ip Detector»

Скрипт для автоматического определения геолокации по IP-адресу и загрузки результатов в облако Яндекс.Диск.

## Описание работы программы
При запуске скрипт последовательно выполняет следующие шаги:
1. Автоматически определяет внешний IP-адрес компьютера.
2. Запрашивает подробные геоданные (город, страну и координаты) для этого IP.
3. Сериализует полученные данные в локальный файл `geo_data.json`.
4. Проверяет наличие папки `IpDetector` на Яндекс.Диске (и создает её при необходимости).
5. Получает временную ссылку для загрузки.
6. Отправляет файл `geo_data.json` в облако.
7. Производит очистку — удаляет временный локальный файл с компьютера.

## Как запустить проект

1. Клонируйте репозиторий.
2. Создайте файл `.env` на основе `.env.example` и укажите свои токены.
3. Установите зависимости:
``` bash
   pip install -r requirements.txt
```
4. Запустите главный файл:
```bash
   python my_01.py
